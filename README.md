# Admas University Library Management System

## 📋 Project Overview
A complete web-based library management system to digitalize Admas University library operations.

## 🎯 Features Implemented
### ✅ User Features:
- User registration with validation
- Secure login system
- Book search functionality
- Book borrowing simulation
- User dashboard with stats
- Responsive design for all devices

### ✅ Technical Features:
- Clean HTML5/CSS3 with modern design
- Interactive JavaScript with form validation
- Local storage for user data persistence
- Font Awesome icons
- Google Fonts integration
- Mobile-responsive design

## 🚀 How to Run
1. Open `index.html` in any modern browser
2. OR use VS Code Live Server extension
3. No server or database required (uses localStorage)
